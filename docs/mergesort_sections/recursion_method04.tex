Relação de Recorrência da mergesort recursiva, $T(n) = 2T(n/2) + n$. Utilizando o método Mestre, temos.
Tome a = 2, b = 2, k = 1. Como $a = b^k \Leftrightarrow 2 = 2^1$, então o T(n) é $\Theta(n^k \log{n}) \Rightarrow \Theta(n \log{n})$
